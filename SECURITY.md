# Security Policy

## Supported Versions

We actively maintain and provide updates for the following versions of the Antigravity Jailbreak rulesets:

| Version | Supported |
| :--- | :--- |
| **v2.0.x** (Active) |   Yes |
| **v1.x.x** |   No |

## Reporting a Vulnerability

If you discover a vulnerability or security issue related to the switcher script or installation routines, please disclose it responsibly:

1. **Contact**: Do not open a public GitHub issue. Send a private message or security report to the repository maintainer.
2. **Details**: Provide a detailed description of the issue, step-by-step instructions to reproduce it, and any potential impacts.
3. **Response**: We will acknowledge receipt of your report within 48 hours and coordinate a patch or mitigation strategy.
